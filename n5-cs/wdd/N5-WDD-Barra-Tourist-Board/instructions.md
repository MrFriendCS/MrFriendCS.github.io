# Barra Tourist Board

## Task

Create a website that the Barra Tourist Board (BTB) can be proud of.  It is to have four pages in total.  Each page is to be linked to the other pages.

Ensure the website has a consistent look and feel.

Each page is to have a theme, such as: location, beaches, activities, transport, accommodation, etc.  Apart from `index.html`, pages should be renamed to the activity.

## Structure

Two folders are required:

* media
* pages

## HTML

### Comments

It is useful to put comments into the code to help explain what is happening, to the next person that works on the code, or to your future self.

Comments are ignored by the web browser, so they are not displayed.  They use `<!--` and `-->`.

``` html
<!-- This is a comment -->
```
