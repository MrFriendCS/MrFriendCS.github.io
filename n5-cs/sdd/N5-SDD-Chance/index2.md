# N5 SDD - Coin Flipper


## Task

Create a small program that will simulate a coin being flipped.  It will use `0` and `1` instead of `Heads` and `Tails`.


### Top level design (structure diagram)

![Structure diagram](assets/sd2.png)


## User Interface


### Example 1

```
Coin Flipper
------------

Zero or One (0 / 1)? 1

Correct!  It was 1.

============
```


### Example 2

```
Coin Flipper
------------

Zero or One (0 / 1)? 0

Incorrect!  It was 1.

============
```
